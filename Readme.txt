                                       ~~~~ Bank Account Manager ~~~~

                                                     by-Vaibhav Haswani

This software is written in c language and compiled using DevC++.
A simple fundamentel of managing bank account is used in this software.


HOW TO USE--

1) You can first add an account to the database by 'open an account option', fill the necessary details 
and your id will be generated kindly note down the generated id.

2) Now you can use various banking options like- ATM service, Check balance, deposit funds & closing a bank account.
